# 大连打滚子 (Dalian Da Gunzi)

大连打滚子单机版 - 基于 JavaFX 的单机卡牌游戏

## 游戏简介

打滚子是大连地区流行的扑克牌游戏，属于"升级"（Sheng Ji）类卡牌游戏。本程序实现了单机版本，玩家可以与三个AI对手进行游戏。

## 功能特色

- **单机对战**：一个人类玩家 + 三个AI对手，组成两支队伍
- **AI难度设置**：支持三种AI智能程度
  - 简单（Easy）：随机出牌
  - 中等（Medium）：基础策略出牌
  - 困难（Hard）：高级策略 + 记牌分析
- **记牌器**：可选开启记牌器，实时显示各花色各牌面的剩余数量
- **完整游戏流程**：叫主、埋底、出牌、计分、升级

## 游戏规则

- 4人游戏，2人一队（玩家与对面AI搭档）
- 使用两副扑克牌（108张，含大小王）
- 5分=5分，10分=10分，K=10分（每局共200分）
- 叫主确定主牌花色，庄家队防守
- 防守方得分 < 80分：庄家队升级
- 防守方得分 ≥ 80分：防守方升级
- 队伍从2级开始，升到A级获胜

## 技术要求

- JDK 21+
- Maven 3.8+

## 构建与运行

```bash
# 构建
mvn clean compile

# 运行测试
mvn test

# 打包
mvn package
```
